Homework course-3

1. Написать свой первый JS код, который будет открывать окна для ввода имени, фамилии и возраста (итого 3 окна), после ввода этой информации необходимо создать объект пользователя и вывести его в консоль.

2. Создать программу, которая запрашивает у пользователя число, в консоль выводит числа от 1 до до этого числа, но пропускает числа, которые кратны (делятся без остатка) 4-м.

2.1 Для продвинутых: - Добавить проверку на тип данных, то есть если передали строку, мы сообщаем об этом пользователю и после этого запускаем самих себя (рекууууурсия...).

2.2 Добавить проверку на размер введенного числа, то есть если пользователь ввел например число более 40, то так же сообщать ему об этом и запустить самих себя (снова рекууууурсия...).

2.3 Создать программу, которая будет каждые 3 секунды увеличивать значение переменной на 3 и выводить итоговое значение в console, а после того, как пройдет 9 секунд, останавливать программу.
